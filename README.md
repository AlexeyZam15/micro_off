## Возможности

🔇 **Быстрое выключение/включение** — один клик или горячая клавиша для управления микрофоном

🎤 **Переключение устройств** — переключайтесь между несколькими устройствами записи на лету

⌨️ **Глобальные горячие клавиши** — управляйте микрофоном из любого приложения

🔄 **Авто-определение** — автоматически определяет текущий активный микрофон

📱 **Адаптивный интерфейс** — правильно масштабируется на разных размерах экрана

🖥️ **Системный трей** — работает свернутым в системном трее с контекстным меню

⚡ **Мгновенная работа** — прямое управление через Windows Core Audio API без задержек

## 🎮 Использование

### Горячие клавиши

| Действие | Горячая клавиша |
|----------|----------------|
| Переключить микрофон | `F4` |
| Выключить микрофон | `F2` |
| Включить микрофон | `F3` |

### Меню системного трея

| Действие | Описание |
|----------|----------|
| Показать | Показать главное окно |
| Выключить | Выключить микрофон |
| Включить | Включить микрофон |
| Выход | Закрыть приложение |

### Главное окно

| Элемент | Описание |
|---------|----------|
| Индикатор статуса | Показывает текущее состояние микрофона (ВКЛ/ВЫКЛ) |
| Кнопки Вкл/Выкл | Ручное управление микрофоном |
| Выбор устройства | Переключение между доступными микрофонами |
| Кнопка обновления | Обновить список устройств |
| Свернуть в трей | Скрыть окно в системный трей |

## 🔧 Настройки программы

Все настройки управляются через интерфейс главного окна:

### Управление устройствами
- **Список устройств** — показывает все доступные устройства записи
- **Авто-выбор** — автоматически выбирает активный микрофон
- **Обновление** — ручное обновление списка устройств

### Интерфейс
- **Адаптивная верстка** — автоматически подстраивается под размер окна
- **Темная тема** — современный дизайн с темно-фиолетовым градиентом
- **Индикаторы статуса** — визуальная обратная связь о состоянии микрофона

## 📂 Структура хранения данных

При первом запуске программа создает следующие файлы в директории приложения:

```

MicroOff/
└── error_log.txt # Файл лога ошибок

```

**Важно:**
- Лог ошибок помогает при отладке, если возникают проблемы
- Все устройства определяются в реальном времени через Windows Core Audio API
- Кэширование не используется — список устройств всегда актуален

## 🔧 Сборка из исходников

### Предварительные требования

1. Установите Python 3.8 или выше (64-bit)
2. Клонируйте репозиторий:

```bash
git clone https://github.com/AlexeyZam15/micro_off.git
cd micro_off
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Сборка исполняемого файла

Запустите скрипт сборки:

```bash
build_exe.bat
```

EXE файл будет создан в папке `dist`.

## 🛠️ Используемые технологии

| Компонент | Назначение |
|-----------|------------|
| **pycaw** | Прямое управление аудиоустройствами через Windows Core Audio API |
| **PyQt5** | Графический интерфейс (GUI) |
| **keyboard** | Глобальные горячие клавиши |
| **comtypes** | Работа с COM-объектами Windows |
| **PyInstaller** | Упаковка в EXE файл |

### Преимущества pycaw перед PowerShell:

- **Мгновенная работа** — без задержек на вызов PowerShell
- **Нет зависимостей** — не требует установки AudioDeviceCmdlets
- **Прямой доступ** — к API Windows без промежуточных слоев
- **Стабильность** — меньше точек отказа
